# PI_005 - POST

Este proyecto corresponde a la práctica **PI_005 - POST**, donde se amplía el prototipo anterior (PI_004) añadiendo rutas de tipo **POST** y probándolas mediante **Postman**.

El objetivo es simular las futuras páginas y formularios de una aplicación estilo **Airbnb para mascotas y cuidadores**.
